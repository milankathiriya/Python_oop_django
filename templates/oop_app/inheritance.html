Simple Inheritance

Code:
class First:
    def hello(self):
        print("HELLO First")


class Second(First):
    def hi(self):
        print("HELLO Second")


o = Second()
o.hello()
o.hi()


print(isinstance(o,Second))
print(issubclass(Second,First))
Output:
HELLO First
HELLO Second
True
True
