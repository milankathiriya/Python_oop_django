Multiple Inheritance

Code:
class First:
    def hello(self):
        print("HELLO First")


class Second:
    def hi(self):
        print("HELLO Second")


class Third(First, Second):
    def py(self):
        print("HELLO Third")



o = Third()
o.hello()
o.hi()
o.py()


print(isinstance(o,Third))
print(issubclass(Second,First))
Output:
HELLO First
HELLO Second
HELLO Third
True
False
