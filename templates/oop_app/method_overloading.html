Method Overloading

Code:
class Person:

    def show(self, name=None):
        if name is not None:
            print("Name : " + name)
        else:
            print("Anonymous.")



p = Person()

p.show()
p.show('Milan')
p.show('Vishal')
Output:
Anonymous.
Name : Milan
Name : Vishal
