Method Overriding

Code:
class Parent:

    def height(self):
        print("Adult height >= 5")



class Base(Parent):

    def height(self):
        print("-----After overriding-----")
        print("Adult height < 5")



a = Parent()
a.height()


c = Base()
c.height()
Output:
Adult height >= 5
-----After overriding-----
Adult height < 5
